CSS Border

Border 테두리

The CSS border properties allow you to specify the style, width, and color of an element's border.
테두리 속성은 요소의 테두리의 색상, 넓이 등 스타일을 구체적으로 설정할 수 있게 해줍니다.
allow - ~할수 있게 해주다, Specify - 구체적으로

  • dotted - 점선 테두리
  • dashed - 대시선 테두리
  • solid - 실선 테두리
  • double - 이중선 테두리
  • groove - 3D 홈 테두리
  • ridge - 3D 돌출된 테두리
  • inset - 3D 들어간 테두리
  • outset - 3D 돌출된 테두리
  • none - 테두리가 없음
  • hidden - 숨겨진 테두리

적용방법
선택자 {border-style: dotted;}

테두리는 4면의 스타일을 다르게 적용할 수 있다.
위 오른쪽 아래 왼쪽의 시계방향 순으로 순차적으로 적용된다.
ex. border-style = "dashed dotted double solid"

한 면만 적용하는 방식도 가능하다.
